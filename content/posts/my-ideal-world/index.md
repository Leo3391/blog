# My Ideal World
These are what I want:
- Everyone should know [how to use the terminal](https://wiwi.blog/blog/why-use-terminal/) ([Tutorial](https://wiwi.video/w/3QdSEB3yJNXQnvv5gJN7ec))
- Many file types are in plain text
- Everyone uses [RSS](http://wiwi.blog/blog/you-should-use-rss/), Email, SMS for messanging
- Everyone uses [Linux](https://wiwi.blog/blog/pianist-using-linux/) on computers and custom ROMs on phones
- Everyone [edits text files]({{< relref "why-you-should-use-vim" >}}) in the terminal
- I can [install apps with APK files](https://keepandroidopen.org/) on Android at anytime as I want
- No [Enshittification](https://wiwi.blog/blog/notebooklm-enshittification/)
