# Tools and Resources I Use

## Development & Terminal Tools
- **Vim** — My go-to text editor for fast and efficient coding
- **Zsh + Oh My Zsh + Powerlevel10k** — Customized shell environment for a powerful terminal experience
- **Termux** — Terminal emulator and Linux environment on Android
- **OpenSSH** — Secure remote connections to servers
- **Git & GitHub** — Version control and code hosting
- **Hugo** — Static site generator I use to build this blog
- **Ranger** — Terminal file manager for quick file navigation
- **w3m** — Terminal web browser for lightweight browsing

## Browsing & Searching
- **DuckDuckGo** — Privacy-focused search engine

## Media & Video Tools
- **yt-dlp** — Command-line tool to download videos
- **mpv** — Lightweight video player
- **NewPipe** — Open-source YouTube app without ads

## Communication
- **Messenger** — Chat app for staying in touch
- **Facebook** — Social networking

## Learning & Productivity
- **ChatGPT** — AI assistant for learning and problem-solving
- **Duolingo** — Language learning app
- **Photomath** — Math problem solver app

## Devices
- **Samsung Galaxy A14** — My primary phone
- **Samsung Tablet A9+** — My tablet for reading and multitasking
